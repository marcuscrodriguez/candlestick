# Candlestick Pattern Certification Exam

Interactive certification platform for learning and assessing foundational candlestick analysis.

## Topics Covered

* OHLC (Open, High, Low, Close)
* Candle Anatomy
* Candle Meaning
* Support
* Resistance
* Confluence
* Trend Recognition
* Pattern Classification
* Context Interpretation

## Candlestick Patterns

1. Engulfing
2. Pin Bar
3. 3-Bar Reversal
4. 3-Bar Continuation
5. Breakout
6. Shrinking Candles

## Educational Philosophy

This project was developed as a distributed-practice learning tool inspired by Dunlosky et al. (2013). The goal is not simply memorization of candlestick patterns but repeated retrieval practice and contextual pattern recognition.

## Technology

* Python
* Streamlit
* Image-based assessments
* Randomized question banks
* Category quotas
